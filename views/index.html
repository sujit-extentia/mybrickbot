MyBrickbot Skill

Do you want to get captivated playing LEGO®? Do you want to be so engrossed that you need not worry about anything but LEGO®?

Well then, welcome to the world of MyBrickBot! A space where Alexa meets LEGO®. Fans can do anything from finding available sets and themes to managing their Brickeset.com accounts.

Love LEGO®. Play LEGO®. Speak Alexa. It's a one stop shop for Alexa users who love LEGO®!

Read More Download Skill

FEATURES

  • Read out all sets available in a user's wish list and own collection
  • Add and remove sets from a users' Brickset.com account
  • Read out specifications about any particular set
  • Read out names of all the themes available on Brickset.com
  • Provide sub-themes for users' existing themes
  • Send error messages to unregistered users
